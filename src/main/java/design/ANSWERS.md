3. Каким образом JAR-файлы этих библиотек попали к нам в проект?
   JAR-файлы библиотек попали в проект автоматически с помощью Gradle. После добавления зависимостей в файл build.gradle
   и обновления проекта Gradle скачал необходимые JAR-файлы из удалённого репозитория и подключил их к проекту.
